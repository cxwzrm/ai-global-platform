## ⚠️ 重要提示

### 开发环境
- Node.js 18+ required
- npm 9+ recommended
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 需要配置的外部服务
1. Sanity.io（内容管理）
2. Vercel（部署）
3. Cloudflare（CDN、安全）
4. Cloudflare R2（图片存储）

### 环境变量
确保在`.env.local`中配置以下变量：
```bash
NEXT_PUBLIC_SANITY_PROJECT_ID=your_project_id
NEXT_PUBLIC_SANITY_DATASET=production
NEXT_PUBLIC_SITE_URL=http://localhost:3000
```
