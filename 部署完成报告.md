## 🌐 部署到Vercel

### 1. 准备Git仓库

```bash
cd ai-global-platform
git add .
git commit -m "Ready for deployment"
git push origin master
```

### 2. Vercel部署

1. 访问 https://vercel.com
2. 点击 "Import Project"
3. 选择GitHub仓库
4. 配置环境变量：
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID=t8o1zoqj
   NEXT_PUBLIC_SANITY_DATASET=production
   NEXT_PUBLIC_SITE_URL=https://your-domain.com
   ```
5. 点击 Deploy

### 3. 域名配置

1. 在Vercel中添加域名
2. 在域名注册商处配置DNS
3. 等待SSL证书自动生成

## 📝 维护说明

### 更新产品数据

直接在Sanity Studio中编辑，或使用API：

```javascript
// 使用Sanity客户端更新
const client = require('@sanity/client')
const sanity = client({
  projectId: 't8o1zoqj',
  dataset: 'production',
  token: 'YOUR_TOKEN',
  useCdn: false
})

// 创建新产品
await sanity.create({
  _type: 'product',
  name: { en: 'New Product' },
  slug: { current: 'new-product' },
  price: 50000,
  // ...
})
```

```bash
node scripts/importData.js
```
