# 🚀 AI Industrial Platform - 快速启动指南

## 第一步：完成外部服务配置

### 1.1 配置Sanity.io CMS

1. **注册Sanity.io账号**
   - 访问 https://www.sanity.io/manage
   - 使用GitHub或Google账号登录
   - 创建新项目

2. **获取项目凭证**
   - 在项目设置中获取：
    - Project ID
    - Dataset名称（默认为production）
   - 创建API Token（用于编辑内容）

3. **更新环境变量**
   ```bash
   cd ai-global-platform
   nano .env.local
   ```
   修改以下值：
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID=your_actual_project_id
   NEXT_PUBLIC_SANITY_DATASET=production
   SANITY_API_TOKEN=your_api_token
   ```

4. **启动Sanity Studio并添加内容**
   ```bash
   npm run sanity
   ```
   - 访问 http://localhost:3333
   - 创建产品分类（Categories）
   - 添加测试产品（Products）

### 1.2 （可选）配置Cloudflare

1. **注册Cloudflare账号**
   - 访问 https://dash.cloudflare.com
   - 添加您的域名
   - 更新域名服务器

2. **配置R2存储（用于图片）**
   - 在Cloudflare Dashboard中创建R2存储桶
   - 配置访问权限
   - 获取R2凭证

### 1.3 （可选）配置Vercel部署

1. **推送代码到GitHub**
   ```bash
   cd ai-global-platform
   # 在GitHub创建新仓库
   git remote add origin https://github.com/yourusername/ai-global-platform.git
   git push -u origin master
   ```

2. **在Vercel中导入项目**
   - 访问 https://vercel.com
   - Import GitHub仓库
   - 配置环境变量
   - Deploy

## 第三步：添加产品数据

### 3.1 在Sanity Studio中添加

1. 启动Sanity Studio：
   ```bash
   npm run sanity
   ```

2. 访问 http://localhost:3333

3. **创建产品分类**：
   - 点击 "Category" → "Create"
   - 填写各语言标题
   - 建议创建：Laser Cutting Machines, Laser Welding Machines, Accessories

4. **创建产品**：
   - 点击 "Product" → "Create"
   - 填写产品信息：
     - Name（各语言）
     - Category（选择分类）
     - Description（各语言）
     - Specifications
     - Price
     - Images
   - 保存

### 3.2 推荐的测试产品数据

#### Laser Welding Machines（激光焊接机）
1. AII-LW300 Manual Welder (500W)

## 第四步：构建和部署

### 4.1 本地构建测试
```bash
npm run build
npm start
```

### 4.2 Vercel部署

1. 在GitHub创建仓库
2. 连接Vercel
3. 配置环境变量：
   ```
   NEXT_PUBLIC_SANITY_PROJECT_ID
   NEXT_PUBLIC_SANITY_DATASET
   NEXT_PUBLIC_SITE_URL
   ```
4. Deploy

### 4.3 自定义域名（可选）

1. 在Vercel中添加域名
2. 在域名注册商处配置DNS
3. 等待SSL证书自动生成

## ❓ 常见问题

### Q: Sanity Studio无法启动？
```bash
# 检查环境变量
echo $NEXT_PUBLIC_SANITY_PROJECT_ID

# 重新安装依赖
rm -rf node_modules package-lock.json
npm install
```

### Q: 多语言切换不工作？
```bash
# 检查middleware.ts配置
# 确保所有语言代码正确：en, de, es, fr, it, nl
```

### Q: 图片无法加载？
```bash
# 检查Sanity项目配置
# 确保images字段正确配置了hotspot
# 检查NEXT_PUBLIC_SANITY_PROJECT_ID是否正确
```

### Q: 部署后页面空白？
- 检查环境变量是否在Vercel中配置
- 确保构建没有错误
- 检查浏览器控制台错误信息
